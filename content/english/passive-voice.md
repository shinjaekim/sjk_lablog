## 수동태 기본 구조

```
능동: Subject + Verb + Object
수동: Object + be + p.p. + (by Subject)
```

## 시제별 수동태

| 시제 | 수동태 형태 | 예시 |
|------|------------|------|
| 현재 | am/is/are + p.p. | The book is read |
| 과거 | was/were + p.p. | The book was read |
| 미래 | will be + p.p. | The book will be read |
| 현재완료 | has/have been + p.p. | The book has been read |

## 수동태를 쓰는 상황

1. **행위자가 불분명할 때**
   - `The window was broken.` (누가 깼는지 모름)

2. **행위자보다 결과가 중요할 때**
   - `The vaccine was developed in 2020.`

3. **공식 문서/학술 글쓰기**
   - `The experiment was conducted under controlled conditions.`

## 주의사항

```
❌ The book was written by he.
✅ The book was written by him.  (전치사 뒤 목적격)
```
