## ~は ~です (A는 B입니다)

```
わたしは がくせい です。
나는 학생입니다.

これは ほん です。
이것은 책입니다.
```

## ~が あります / います (있습니다)

```
つくえの うえに ほんが あります。
책상 위에 책이 있습니다. (사물)

こうえんに いぬが います。
공원에 개가 있습니다. (생물)
```

## ~て ください (해 주세요)

```
すわって ください。
앉아 주세요.

みて ください。
봐 주세요.
```

## ~ませんか (하지 않겠습니까?)

```
いっしょに たべませんか。
같이 먹지 않겠습니까?
```

## 시간 표현

| 일본어 | 한국어 |
|--------|--------|
| いま なんじ ですか | 지금 몇 시입니까? |
| ごじ はん です | 5시 반입니다 |
| きのう / きょう / あした | 어제 / 오늘 / 내일 |
